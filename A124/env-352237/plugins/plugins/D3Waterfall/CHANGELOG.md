# Change Log
All notable changes to this project will be documented in this file.

 ## [0.0.1] - 2016-09-07
### Added
- You can have labels in the bars showing the values of the bar
- You can remove the Y Axis
- You can set a color for negative and positive values a specific metric
- You can specify a color for a specific metric
- You can start the first metric with a specific value

## [0.0.2] - 2016-11-23
### Added
- D3Waterfall now supports exporting to PDF.

## [0.0.3] - 2017-01-13
### Fixed
- Fixed the issue that total labels is not formatted like the other metrics

## [0.0.4] - 2018-01-02
