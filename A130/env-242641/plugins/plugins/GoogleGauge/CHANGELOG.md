# Change Log
All notable changes to this project will be documented in this file.

## [0.0.1] - 2016-08-08
### Changed
- The ratio of currentValue/maxValue was used instead of the actually value, so that the gauges are displayed properly.

### Added
- Custom properties were added to allow the users to modify the low, medium and high bands ranges and their corresponding colors.

## [0.0.2] - 2016-11-23
### Added
- GoogleGauge now supports exporting to PDF.

## [0.0.3] - 2018-3-19
### Changed
- Updating style to load icons on RSD configuration page

## [0.0.4] - 2018-6-28
### Added
